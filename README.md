# 🌐 HTML Course – Simple to Intermediate

Welcome! This repo teaches you HTML step by step with hands-on exercises.

## 📁 Folder Structure

| Folder | Topic | Level |
|--------|-------|-------|
| ex1 | Basic HTML Structure | ⭐ Beginner |
| ex2 | Headings & Paragraphs | ⭐ Beginner |
| ex3 | Links & Images | ⭐ Beginner |
| ex4 | Lists (ul, ol, dl) | ⭐ Beginner |
| ex5 | Tables | ⭐⭐ Beginner-Mid |
| ex6 | Forms & Inputs | ⭐⭐ Beginner-Mid |
| ex7 | Semantic HTML | ⭐⭐ Intermediate |
| ex8 | Div, Span & Inline vs Block | ⭐⭐ Intermediate |
| ex9 | HTML5 Media (Audio & Video) | ⭐⭐ Intermediate |
| ex10 | Mini Project – Profile Page | ⭐⭐⭐ Intermediate |

## ▶️ How to Use

1. Open any folder (start from `ex1`)
2. Read the `README.md` inside it first
3. Open the `.html` file in your browser
4. Try editing the code yourself!

> **Tip:** Use [VS Code](https://code.visualstudio.com/) + Live Server extension to see changes instantly.
