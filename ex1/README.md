# Exercise 1 – Basic HTML Structure ⭐

## 🎯 What You Will Learn
- What is an HTML file
- The basic skeleton every HTML page needs
- What each tag does

---

## 📖 Explanation

Every HTML page has this skeleton:

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Page Title</title>
</head>
<body>

  <!-- Your content goes here -->

</body>
</html>
```

### Tag Breakdown

| Tag | What it does |
|-----|-------------|
| `<!DOCTYPE html>` | Tells the browser "this is an HTML5 file" |
| `<html>` | Root of the entire page |
| `<head>` | Hidden info (title, settings) – not shown on page |
| `<meta charset="UTF-8">` | Supports all characters (emojis, Tamil, etc.) |
| `<title>` | Text shown on the browser tab |
| `<body>` | Everything visible on the page goes here |
| `<h1>` | A big heading |
| `<p>` | A paragraph |
