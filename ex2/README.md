# Exercise 2 – Headings & Paragraphs ⭐

## 🎯 What You Will Learn
- All 6 heading sizes
- Paragraph tag
- Text formatting (bold, italic, underline)
- Line break and horizontal rule

---

## 📖 Explanation

### Headings (h1 to h6)
```html
<h1>Biggest</h1>
<h2>Second biggest</h2>
<h3>Third</h3>
<!-- ... up to h6 (smallest) -->
```
Use **only one `<h1>`** per page (it's like the main title).

### Paragraphs
```html
<p>This is a paragraph.</p>
```

### Text Formatting Tags

| Tag | Result |
|-----|--------|
| `<b>text</b>` | **bold** |
| `<i>text</i>` | *italic* |
| `<u>text</u>` | underlined |
| `<strong>text</strong>` | bold (semantic) |
| `<em>text</em>` | italic (semantic) |
| `<br>` | line break (no closing tag needed) |
| `<hr>` | horizontal line separator |

- Use headings in order: h1 → h2 → h3 (don't skip)
